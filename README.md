# Make SVG Graph from formulas

## Running without generating SVG file

`./svggraph`

## Running with SVG file

Replace W, H, X, Y with numbers.

`./svgraph output.svg W H X Y`

## Supported functions

Replace a, b, c with numbers.

- `y = a x`
- `y = x + a`
- `y = x - a`
- `y = sin a`
- `y = cos a`
- `y = tg a`
